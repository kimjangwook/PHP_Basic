# PHP 問題・回答

プログラミング言語であるPHPを学習する人々のため、理解を深めることができるよう宿題を作りました。

## 使い方

追加される宿題のリスト・ハッシュコードを更新します。

ハッシュコードを確認しながら下記方法を使ってください。

<strong>初めての方</strong>

```shell script
$ git clone https://github.com/kimjangwook/PHP_Basic.git
$ cd PHP_Basic
$ git checkout <hash-code>
```

<strong>2回目からのの方</strong>

```shell script
$ cd PHP_Basic
$ git pull
$ git checkout <hash-code>
```
